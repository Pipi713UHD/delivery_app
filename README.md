# Delivery App

Aplicación móvil de delivery desarrollada con Flutter para el frontend y PHP/MySQL para el backend.

## Características principales

- Autenticación y gestión de usuarios.
- Roles de Administrador, Cliente, Restaurante y Repartidor.
- Gestión de restaurantes y categorías.
- Administración de productos por restaurante.
- Carrito de compras.
- Creación y gestión de pedidos.
- Gestión de direcciones con integración de Google Maps.
- Selección de ubicación exacta mediante coordenadas geográficas.
- Gestión de repartidores.
- Asignación y seguimiento de pedidos.
- Panel administrativo para control de usuarios, restaurantes, productos y pedidos.
- API REST desarrollada en PHP.
- Base de datos MySQL relacional.

### Frontend
- Flutter
- Dart

### Backend
- PHP
- MySQL
- XAMPP

### Servicios externos
- Google Maps Platform
- Google Maps Flutter

## Objetivo del proyecto

Desarrollar una plataforma de entrega de comida que permita conectar clientes, restaurantes y repartidores en un mismo sistema, facilitando la gestión de pedidos y el seguimiento de entregas mediante geolocalización.
